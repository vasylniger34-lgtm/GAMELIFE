## Game Life (PWA)

**Game Life** — це мобайл‑фьорст PWA‑додаток українською мовою для гейміфікованого ведення дня, квестів, проєктів та звичок. Працює офлайн, зберігає дані локально і може встановлюватися як застосунок.

### Запуск локально

1. Встановіть залежності:

```bash
npm install
```

2. Запустіть dev‑сервер:

```bash
npm run dev
```

Після цього відкрийте адресу, яку покаже Vite (зазвичай `http://localhost:5173`).

3. Зібрати продакшн‑версію:

```bash
npm run build
npm run preview
```

### Основні екрани

- **Головна** — поточна дата/час, стати, енергія, кнопка **«Новий день»**, швидкі дії.
- **Сьогодні** — квести сьогоднішнього дня, відмітка виконання/провалу.
- **Планер** — планування квестів на сьогодні, завтра та майбутні дні.
- **Проєкти** — головні квести, об’єднані в проєкти з прогрес‑баром.
- **Щоденний підсумок** — резюме останнього завершеного дня.
- **Статистика** — загальна аналітика по днях, квестах, стріках.
- **Архів** — історія завершених днів та квестів (тільки читання).

### Приклади даних

При першому запуску в додатку вже будуть:

- один демонстраційний проєкт **«Головний проєкт місяця»**;
- кілька прикладових квестів на сьогодні (звичка, головний, побічний).

Всі дані (дні, квести, стати) зберігаються у `localStorage` через Zustand‑стан. Додаток автоматично:

- прив’язує дії до поточної дати;
- активує заплановані на сьогодні квести;
- фіксує початок та завершення дня;
- обмежує дії за енергією.

### PWA

- Використовується `vite-plugin-pwa` з **авто‑оновленням** service worker.
- Додаток кешує shell та статику, працює офлайн (на рівні браузера).
- Є `manifest.webmanifest` для встановлення на домашній екран.
